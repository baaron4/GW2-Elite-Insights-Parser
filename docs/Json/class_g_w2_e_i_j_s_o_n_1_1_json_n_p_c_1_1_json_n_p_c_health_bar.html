 Health value for when the bar is at 100%.

bool Active
 Indicates that, at the end of the encounter, this health bar was the active one.
JsonNPC.HealthPercentBurned and JsonNPC.BarrierPercent operates on the active health bar.
Health bars before active health bar have been fully consumed, health damage done computed as JsonNPCHealthBar.Health * (JsonNPCHealthBar.MaxPercent - JsonNPCHealthBar.MinPercent) / 100.0.
Health bars after active health bar are untouched, remaining health computed as JsonNPCHealthBar.Health * (JsonNPCHealthBar.MaxPercent - JsonNPCHealthBar.MinPercent) / 100.0.
Remaining health on active health bar is computed as JsonNPCHealthBar.Health * (100.0 - JsonNPC.HealthPercentBurned - JsonNPCHealthBar.MinPercent) / 100.0.
